Manual para poder llevar a cabo la correcta ejecución del código: 

Primero requerimos de tener la máquina VisualCodeStudio, y usamos el lenguaje Python.

Segundo debemos tener instalados las librerías de: Flet, Fpdf, Matploblib, Json, Os, Datetime.


Luego de esto tenemos definido los requerimientos que debe cumplir el sistema: 

Primero esta la Gestion de Pedidos para las mesas: donde se le pueda tomar el pedido al usuario con indicaciones especificas.

Segundo la Gestion de Cobro y Facturación: donde se puedan manejar el flujo de las facturas emitidas tanto a los consumidores como a los proveedores.

Gestion de Reserva: donde se pueda reservar una mesa con antelación y poder asignarlas también.

Gestion de Inventario: donde se gestiona el stock disponible.
